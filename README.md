# NCM 转 MP3 转换器

一个纯浏览器端的网易云音乐 NCM 格式解密转换工具，支持批量转换、封面显示和音频预览。

## 功能特性

- 🔐 **纯浏览器端解密** - 所有操作在本地完成，无需上传文件，保护隐私
- 📦 **批量转换** - 支持同时转换多个 NCM 文件
- 🖼️ **封面显示** - 自动提取并显示专辑封面图片
- 🎵 **音频预览** - 内置播放器，转换后可直接预览
- 📥 **打包下载** - 支持将转换后的文件打包为 ZIP 下载
- 🏷️ **ID3 标签** - 自动写入歌曲名称、艺术家、专辑、封面等元数据

## 支持的格式

NCM 文件解密后可能输出以下格式：
- MP3
- FLAC
- M4A
- OGG

## 使用方法

1. 打开 `index.html` 文件（需要通过 HTTP 服务器访问）
2. 拖拽 NCM 文件到页面，或点击选择文件
3. 点击"开始转换"按钮
4. 转换完成后可以：
   - 点击"预览"按钮播放音频
   - 点击"下载"按钮下载单个文件
   - 点击"打包下载"按钮下载所有转换成功的文件

## 本地运行

1. **直接打开** - 双击 `index.html` 文件在浏览器中打开即可使用（部分浏览器可能需要允许本地文件访问）

2. **通过 HTTP 服务器**（推荐，功能更完整）：

```bash
# 使用 Python
python -m http.server 8080

# 或使用 Node.js
npx http-server -p 8080
```

然后在浏览器中访问 `http://localhost:8080`

## 技术实现

- **AES-128-ECB 解密** - 用于解密 NCM 文件的密钥和元数据
- **CR4 流加密** - 用于解密音频数据
- **ID3v2 标签** - 用于写入 MP3 元数据和封面图片
- **JSZip** - 用于打包下载功能
